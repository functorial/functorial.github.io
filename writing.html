Publications

  • PureScript By Example - An introductory book on functional programming for Javascript developers. It covers many core FP concepts, providing real-world examples as motivation, including callback-based programming, canvas graphics and DOM templating.
  • Declarative UIs Are The Future - And the Future Is Comonadic! - A short introduction to a comonadic semantics for user interfaces, or alternatively, a study of existing UI patterns using the language of category theory.
  • Stack Safety For Free! - A summary of some techniques developed in PureScript (and also used in Scala) in order to make some functional idioms expressible without exhausting the stack.
